## 🚀 Projects

### 🛰️ Deep Transfer Learning for Multi-Class Semantic Segmentation of Satellite Images
`Dec 2025 – Jan 2026` · **🥈 2nd Prize, Aakar Project Expo, VBIT**

Developed and fine-tuned a deep transfer learning model for semantic segmentation of satellite imagery using pretrained CNNs. Enabled accurate pixel-level land cover classification with performance optimization through evaluation metrics.

**Tech:** Python · Deep Learning · CNNs · Transfer Learning · Semantic Segmentation

---

### 🔐 Smart Attendance System — RFID + Fingerprint + Gmail Automation
`Jan 2025 – Jun 2025` · **IEEE International Conference Publication (Oct 2025)**

Raspberry Pi-based dual-authentication attendance system integrating RFID and biometric fingerprint verification with Python control logic, SQLite offline database, and Gmail API automated notifications. Achieved **98.5% authentication accuracy**.

**Tech:** Raspberry Pi · Python · SQLite · Gmail API · RFID · Biometric · Embedded Systems

---

### 🔲 RTL to GDSII Mini Flow
`Sep – Dec 2025`

Implemented a basic RTL-to-GDSII ASIC design flow covering synthesis, placement, clock tree synthesis, routing and timing analysis using the Cadence tool environment.

**Tech:** Cadence Innovus · Cadence Virtuoso · STA · ASIC Flow

---

### ⚙️ 16-bit ALU Using Verilog HDL
`Jul – Sep 2025`

Designed and simulated a 16-bit Arithmetic Logic Unit supporting arithmetic and logical operations. Developed RTL modules and verified functionality through testbench simulation.

**Tech:** Verilog HDL · Xilinx Vivado · RTL Design · Testbench

---

### 🌍 Earthquake Detector Using Arduino UNO
`Jul – Dec 2024`

Real-time seismic activity detection system using Arduino Uno and ADXL335 accelerometer. Calibrated vibration thresholds to distinguish genuine seismic events from environmental noise, with live LCD alerts.

**Tech:** Arduino Uno · ADXL335 · Embedded C · Sensor Calibration
